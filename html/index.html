GIT Training

I will train GIT commands here

  • git clone |path to repository| ➜ копирует репозиторий в указанную папку
  • git cd |folder name| ➜ меняет папку, в которой работает git-bash
  • git status ➜ показывает, были ли изменения в указанной папке
  • git add |file/folder name| ➜ добавляет указанные файлы в локальный репозиторий
  • git config --global user.email
  • git config --global user.name
  • git commit -am "list of changes" ➜ добавляет указанные файлы в локальный репозиторий
  • git push ➜ отправляет файлы с изменениями на сервер
